# 🤖 AI-Powered Interview Coach

An AI-based web application that simulates real interview scenarios, evaluates candidate responses, and provides structured feedback using Generative AI.

---

## 🚀 Features

- 🎯 Role-based mock interviews (Data Analyst, ML Engineer, etc.)
- 💬 AI-generated interview questions
- 🧠 Intelligent answer evaluation
- 📊 Feedback includes:
  - Technical Accuracy Score
  - Grammar Score
  - Confidence Score
  - Strengths & Weaknesses
  - Improvement Suggestions
- 🎤 Voice input support (optional)
- 🔁 Dynamic question generation

---

## 🛠 Tech Stack

- **Frontend/UI:** Streamlit  
- **Backend:** Python  
- **LLM API:** Google Gemini API  
- **Voice Processing:** SpeechRecognition (optional)  
- **Environment Management:** python-dotenv
